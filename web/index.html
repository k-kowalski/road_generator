About This Project

Road Generator is a procedural sandbox built around a shared C++ core for polyline authoring, subdivision, cleanup, and ribbon mesh generation.

The same core solvers and algorithms feed both the native DirectX 12 app and this WebAssembly + WebGPU demo.

Geometry programming in C++/GPU allows for high performance, low-level control over the generation process and portability across platforms.
